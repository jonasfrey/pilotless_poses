What's available on image?
  • image.source — full path to the source image
  • image.fileName — just the file name
  • image.width, image.height — pixel dimensions
  • image.peopleCount — number of detected people
  • image.people — array of { id, keypoints: [{ name, x, y, confidence }] }
  • image.pose — the full raw pose JSON
  • image.keypoint(name, personIndex = 0) — helper to fetch one keypoint
